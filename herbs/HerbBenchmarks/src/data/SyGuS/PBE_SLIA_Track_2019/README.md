# PBE SLIA Track 2019

String-manipulation problems from the SyGuS Challenge 2019 using programming by example. Inputs and output (see `data.jl`) are strings which need to be reached via a sequence of string-operations (see `grammars.jl`). 
We combined problems from the `euphony` and `from_2018` track into one benchmark.

Problems and their corresponding grammars share the same suffix.
